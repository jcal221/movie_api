API Documentation

Movie API Endpoints

Operation Request Method URL Endpoint Response
Return a list of ALL movies to the user GET /movies JSON array containing data of all movies
Return data about a single movie by title to the user GET /movies/{title} JSON object containing data about the movie with the specified title
Return data about a genre by name/title GET /genres/{name} JSON object containing data about the genre with the specified name/title
Return data about a director by name GET /directors/{name} JSON object containing data about the director with the specified name
Allow new users to register POST /users/register Text indicating the successful registration of a new user
Allow users to update their user info (username) PUT /users/{userId} Text indicating the successful update of the user's information

Endpoint Documentation

  • /movies
    • Method: GET
    • Query Parameters: None
    • Response: JSON array
  • /movies/{title}
    • Method: GET
    • Query Parameters: None
    • Response: JSON object
  • /genres/{name}
    • Method: GET
    • Query Parameters: None
    • Response: JSON object
  • /directors/{name}
    • Method: GET
    • Query Parameters: None
    • Response: JSON object
  • /users/register
    • Method: POST
    • Request Body: JSON object
    • Example Request Body: 
                  {
              "username": "john_doe",
              "email": "john.doe@example.com",
              "password": "password123"
            }
    • Response: Text
  • /users/{userId}
    • Method: PUT
    • Request Body: JSON object
    • Example Request Body: 
                  {
              "username": "new_username"
            }
    • Response: Text
